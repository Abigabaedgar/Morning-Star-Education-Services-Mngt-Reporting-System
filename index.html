📋 Expected file format
• Same format as MID_TERM_ASSESMENT.xlsx
• Class header row: JY1 MID TERM ONE 2026
• Column headers: NO · NAME · ENG · MTC · LIT · READ · RE · SCI · SST
• One file can contain multiple classes (sheets or stacked sections)
• Upload class by class or all at once
⚙️ How import works
• System detects class & subjects automatically
• New students → added to registry
• Existing students → scores updated (matched by name)
• You preview everything before confirming
• Supports Midterm, BOT, and End-of-Term files
